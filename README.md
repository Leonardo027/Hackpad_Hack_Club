## About The Project
This is a fully custom mechanical macropad designed from scratch as part of the Hack Club Hackpad initiative. The goal was to learn PCB design (KiCad), CAD modeling (Onshape), and firmware configuration (KMK).

It features a 3x3 switch matrix (9 keys) and dual rotary encoders for volume/media control, all running on a Seeed XIAO RP2040 microcontroller. The case was designed to be easily 3D printed.

**Tech Stack:**
* **PCB:** KiCad 8.0
* **CAD:** Onshape (Tray case design)
* **Firmware:** KMK (CircuitPython)
* **MCU:** Seeed XIAO RP2040
